# Блогикум (django_sprint4)

Финальная версия учебного проекта «Блогикум» на Django.

В проекте реализованы:
- публикации, категории, профили пользователей;
- регистрация и авторизация;
- создание, редактирование и удаление постов;
- комментарии к постам;
- пагинация;
- загрузка изображений к постам;
- кастомные страницы ошибок `403/404/500`;
- админ-панель для управления контентом.

## Быстрый запуск (одной командой)

Из корня репозитория:

```bat
start_forum.bat
```

Что делает команда:
1. Выполняет миграции.
2. Заполняет демо-данные.
3. Запускает сервер на `127.0.0.1:8000`.

Альтернативный запуск без `.bat`:

```bash
cd blogicum
python manage.py start_demo
```

## Основные URL для показа преподавателю

- главная: `http://127.0.0.1:8000/`
- админка: `http://127.0.0.1:8000/admin/`
- вход: `http://127.0.0.1:8000/auth/login/`
- регистрация: `http://127.0.0.1:8000/auth/registration/`

## Краткий сценарий защиты

1. Запустить проект командой `start_forum.bat`.
2. Открыть главную страницу и показать пагинацию, категории и карточки постов.
3. Зайти в админку под `admin/admin12345` и показать модели: `Post`, `Category`, `Location`, `Comment`.
4. Выйти из админки, зарегистрировать нового пользователя через `/auth/registration/`.
5. Зайти под обычным пользователем и создать пост на `/posts/create/`.
6. Открыть созданный пост, добавить комментарий, затем отредактировать и удалить комментарий.
7. Показать профиль пользователя `/profile/<username>/` и список его публикаций.

## Работа с демо-данными

Команда `seed_demo`:
- очищает старые демо-записи;
- очищает папку `blogicum/media/posts_images`;
- заново создаёт демо-посты, комментарии и изображения.

Запуск вручную:

```bash
cd blogicum
python manage.py seed_demo
```
